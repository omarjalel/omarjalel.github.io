Data Integration between MQTT broker, Modbus TCP and Azure Cloud (IoT Hub)

This Demo includes 2 parts:

Part 1 : MQTT Broker - Azure Cloud connectivity

  • Setting Up ESP32 with DHT11 Sensor to read the temperature and the humidity...
  • Send the sensor data to MQTT Broker hosted locally
  • Create Client1 with C# to forward all the telemetries from MQTT Broker to Azure IoT Hub
  • Use Data Storage in Azure to archive and process the received data

Part 2 : Modbus TCP (Siemens PLC) - Azure Cloud connectivity

  • Setting Up a local Modbus TCP Server
  • Setting Up Siemens PLC-1511 Simulation and create a Modbus TCP Client in Tia Portal
  • Write Switch ON and OFF 2 Coils, Write into 2 holding registers with 2 MB Clients
  • Create Client2 with C# to forward all the telemetries from Modbus TCP Server to Azure Device Provisioning Service (DPS) then to Azure IoT Hub
  • Archive and process the received data in Azure Cloud
