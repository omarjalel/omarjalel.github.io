Simulating Robot Operating System on Ubuntu VM (Azure VM) - Data forward to NODE-RED

Project description : Deploy a Robot Operating System (ROS) simulation environment under Ubuntu OS with Gazebo and the ROS-Noetic package, all hosted Microsoft Azure Cloud VM.

  • Deploying a Virtual Machine into Azure Cloud
  • Setting Up the VM Network Access to be accessible from Remote Desktop through SSH
  • Simulating the Robot Operating System (ROS) under Ubuntu with Gazebo and ROS-Noetic package
  • Launching ROS-Bridge to expose data via WebSocket
  • Using Node-RED to connect to the ROS-Bridge server and read object coordinates in real time
